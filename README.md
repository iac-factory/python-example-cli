# Python Example CLI

## Usage

###### Dependencies

```bash
python -m pip install --upgrade pip build setuptools
```

###### Build

```bash
python -m build
```

###### Install

```bash
python -m pip install .
```

###### Usage

```bash
python-example-cli
```
